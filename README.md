<h2>Description</h2>

<p>
This machine learning project predicts passenger survival on the Titanic using the Kaggle Titanic dataset. The project involves data preprocessing, feature engineering, and model training with logistic regression. Evaluation metrics such as Accuracy, AUC, and the Confusion Matrix are used to assess model performance.
</p>
